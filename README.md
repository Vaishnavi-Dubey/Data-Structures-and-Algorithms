# 🧬 Data Structures & Algorithms

> A comprehensive repository documenting my journey through computer science fundamentals, focusing on Object-Oriented Programming (OOP) and classical Data Structures & Algorithms in Java and C++.

This repository serves as a personal study guide and implementation library for core CSE concepts, ranging from basic syntax to complex problem-solving patterns.

---

## 📚 Contents

### ☕ Java: Object-Oriented Programming (OOP)
Detailed implementations of core OOP principles in Java:
- **Abstraction**: Using abstract classes and interfaces (`OOPS_Abstraction.java`, `OOPS_Interface.java`).
- **Encapsulation**: Proper use of access modifiers and `getters/setters` (`OOPS_encapsulation.java`, `OOPS_gettersSetters.java`).
- **Inheritance**: Models of code reusability (`OOPS_Inheritance.java`).
- **Polymorphism**: Compile-time and runtime polymorphism examples (`OOPS_Polymorphism.java`).
- **Constructors**: Deep dive into constructor overloading and types (`OOPS_Constructors.java`).

### 🧩 Java: Data Structures & Algorithms
Implementations and solutions to classic algorithmic problems:
- **ArrayLists**: Custom implementations and usage patterns.
- **Recursion**: Solving complex problems through recursive decomposition.
- **Two-Pointer Technique**: Optimized solutions for problems like `MaxWaterContainer.java`, `PairSumSortedArray.java`, and `PairSum2.java`.
- **Patterns**: Logical building blocks through coordinate-based patterns.

### 💻 C++: The Basics
Foundational C++ code snippets located in `CppBasics/`:
- **Data Types**: Understanding memory and primitive types.
- **I/O Operations**: Basic input/output handling in C++.

## 🚀 How to Use

Each file is a standalone Java/C++ program. To run any implementation, ensure you have the JDK (for Java) or a C++ compiler (like G++) installed.

### Run a Java file:
```bash
javac FileName.java
java FileName
```

### Run a C++ file:
```bash
g++ inputOutput.cpp -o output
./output
```
