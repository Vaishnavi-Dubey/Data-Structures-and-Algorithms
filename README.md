## ✨ Key Features

- ☕ **Complete OOP Suite** — Abstraction, Encapsulation, Inheritance, Polymorphism, Constructors, Interfaces
- 📊 **Data Structures** — ArrayLists, custom implementations, and collection operations
- 🔁 **Recursion** — Recursive decomposition and backtracking patterns
- 🎯 **Two-Pointer Technique** — Optimized O(n) solutions for pair-sum and container problems
- 🧩 **Pattern Problems** — Coordinate-based pattern generation for logical thinking
- 💻 **C++ Fundamentals** — Data types, I/O operations, and C++ basics
- ⚡ **Optimized Algorithms** — Focus on time complexity reduction and efficient approaches

## 🏗️ Architecture / How It Works

### Learning Path

```
OOP Fundamentals → Data Structures → Algorithmic Patterns → Optimization
```

### OOP Hierarchy Implemented

```
OOPS.java (Base Concepts)
├── OOPS_encapsulation.java      # Access modifiers, data hiding
├── OOPS_gettersSetters.java     # Property accessors
├── OOPS_Constructors.java       # Constructor overloading
├── OOPS_Inheritance.java        # Code reusability
├── OOPS_Polymorphism.java       # Runtime & compile-time
├── OOPS_Abstraction.java        # Abstract classes
├── OOPS_Interface.java          # Interface contracts
└── oops_static.java             # Static members & methods
```

### Algorithmic Patterns

| Pattern | Problem | Complexity |
|---------|---------|-----------|
| **Two-Pointer** | MaxWaterContainer | O(n) |
| **Two-Pointer** | PairSumSortedArray | O(n) |
| **Two-Pointer** | PairSum2 (unsorted) | O(n log n) |
| **Recursion** | Recursive problem set | Varies |
| **Patterns** | Coordinate-based patterns | O(n²) |

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Vaishnavi-Dubey/Data-Structures-and-Algorithms.git
cd Data-Structures-and-Algorithms

# Ensure JDK is installed
java --version
```

## ▶️ Usage

### Run a Java file:
```bash
javac MaxWaterContainer.java
java MaxWaterContainer
```

### Run a C++ file:
```bash
cd CppBasics
g++ inputOutput.cpp -o output
./output
```
